## 👨‍🔬 About Me

I am an AI/ML Engineer and Researcher specializing in **generative AI, multi-agent orchestration, and distributed data systems**. I bridge the gap between low-level architectural engineering and high-stakes machine learning deployments.

* 🎓 **Education:** Pursuing an Integrated M.Tech (5-Year MEng equivalent) in Computational & Data Science at Vellore Institute of Technology (Class of 2027).
* 🔭 **Current Focus:** Building deterministic, multi-agent LLM architectures (LangGraph) and researching biologically-inspired state-space models (Spiking NNs).
* 🛰️ **Industry Experience:** Ex-ISRO AI/ML Researcher (engineered LSTM telemetry-forecasting models for PSLV/SSLV). Currently validating production-grade frontier LLM pipelines and agentic workflows as a freelance AI Specialist.
* ⚡ **Core Competencies:** PyTorch, C++ systems programming, Distributed Big Data (PySpark), Generative AI, and Mechanistic Interpretability.

| **[Synapse.cpp](https://github.com/TechieSamosa/Synapse.cpp)** | A Multi-Layer Perceptron built entirely from scratch in **pure C++17**. Implements manual memory management, matrix algebra, Adam optimizer, and backpropagation without any external libraries. | `C++17` `Linear Algebra` `Neural Networks` |

| **[Nexus Scout](https://github.com/TechieSamosa/nexus-scout-catalyst)** | A deterministic multi-agent HR orchestrator. Uses LangGraph, batch-chunking inference to bypass API limits, and strict Pydantic JSON routing for automated talent evaluation. | `LangGraph` `Llama-3` `Streamlit` |
